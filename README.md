## 项目1 游乐项目票价
这个代码会**根据游客的情况输出对应的票价**。首先，程序会要求游客以阿拉伯数字的形式输入自己的年龄。
考虑到安全问题，对于14岁及以下、60岁及以上的人群，程序会输出"抱歉，您的年龄不符合安全要求"。
对于符合要求的游客，在用户按要求输入自己的游玩情况后，程序则会输出票价。
```
age = int(input("您的年龄是："))
while True:
    if age <= 14 or age >= 60:
        print("抱歉，您的年龄不符合安全要求")
        break
    # 排除不符合年龄要求的游客
    tour = input("请输入 单次游玩 或 一次往返：")
    # 符合年龄要求的顾客才需要再输入游玩情况
    if tour == '单次游玩':
        print('您的票价为50元')
    elif tour == '一次往返':
        print('您的票价为90元')
        break
    else:
        print('输入不符合要求，请重新输入 单次游玩 或一次往返：')
```

## 项目2 等差数列求和
这个代码会**根据等差数列的首项、公差、以及第n项输出等差数列从第1项加到第n项的和**。
在用户按要求输入首项a、步长d和n的值之后，程序会使用for循环进行遍历，计算每一项的值并进行累加。
```
# 根据起始项a₁和步长值，可以对等差数列求和，即a₁+ a₂+……+an的值
n = int(input("请输入n的值（n为整数）："))
d = int(input("请输入步长d的值："))
a = int(input("请输入起始项a₁的值："))
# 输入n、d、a₁值
sum_range = 0
# 设置储存列表之和的变量，并预设为0
for x in range(1,n+1):
    an = a+(x-1)*d
    sum_range += an
# 将遍历的值累加到预设的变量
print("a₁+ a₂+……+an的值为",sum_range)
# 打印最后的结果
```
